# 二叉树  
## 1.求树的最大深度  
本体主要是使用递归的思想。  
## 2.从前序和中序遍历序列构造二叉树  
这道题主要是找到前序中的根节点，然后利用存储的map中的值找到中序遍历中根节点。然后确定前序和中序的左右子树，利用左右子树递归实现二叉树。
